# Skill: вскрытие хранилища

Хранилище защищено собственным шифром. Тип шифра:
- easy: 3-значный код (брутфорс среди 10..999, единственный валидный по hint)
- medium: 5-значный код + 2 ограничения
- hard: каскад: caesar → digits

## Алгоритм

1. Прочитать `cipher_spec.type` и применить соответствующий метод (см. cipher_solving.md).
2. Hacker должен физически быть на клетке `vault`.
3. Попыток: 3 (easy), 2 (medium), 1 (hard). Промах = `alarm` сразу (Фаза 1).
