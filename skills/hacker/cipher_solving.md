# Skill: решение шифров

Шифр приходит в `HackRequest.payload.cipher_spec`. Решение возвращается в `HackResponse.payload.answer`.

## Caesar (easy)

1. Посчитать частоты символов.
2. Найти сдвиг, при котором результат — связный текст из словаря.
3. Если есть выбор между несколькими, предпочесть короткий и тематический (банк, замок, деньги).

Чек-лист:
- Алфавит: RU (33) или EN (26).
- Сдвиг: 1..|alphabet|-1.
- Регистр результата: верхний.

## Sudoku 4×4 (medium)

1. Разделить сетку 4×4 на четыре блока 2×2.
2. Применить ограничения: уникальность по строке, столбцу, блоку.
3. Использовать метод исключений + backtracking.
4. Решение **единственно** — генератор это гарантирует.

## Logic puzzle (hard)

1. Прочитать ограничения из `spec.rules`.
2. Перебором (≤10000 вариантов) или редукцией найти единственный набор переменных.
3. Сначала записать ограничения формально, потом перебирать.

## Формат ответа

Только JSON:
```json
{
  "answer": "<value>",
  "reasoning": "<≤2 предложения>",
  "confidence": 0..1
}
```

## При неудаче

- Перепроверить parse `cipher_spec`.
- Если confidence < 0.5 — лучше потратить попытку на «безопасный» candidate, чем выдать что попало.
