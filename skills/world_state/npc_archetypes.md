# Skill: отыгрыш NPC

При встрече с NPC роль World State Agent — отыгрывать его реалистично, **не выдавая** Robber'у архетип явно. Архетип передаётся в `EncounterEvent.payload.archetype`, но реплика NPC должна быть согласована с архетипом, не просто метка.

## aggressive
- «Стой! Я вызвал полицию! Что вы тут делаете?»
- «Не подходи, у меня тревожная кнопка!»
- Жесты: тянется к телефону, перекрывает проход.

## scared
- «Пожалуйста, не убивайте меня. Я ничего не видел…»
- «Я отдам всё, у меня дети.»
- Дрожит, отступает, прикрывается руками.

## neutral
- «А, инкассация? Я не в курсе расписания…»
- «Подождите, мне нужно отойти.»
- Не оценивает угрозу, может попытаться обойти и пойти своей дорогой.

## Формат вывода
```json
{
  "npc_id": "npc_3",
  "archetype": "aggressive|scared|neutral",
  "utterance": "<реплика NPC, 1-2 предложения>",
  "body_language": "<короткое описание невербалики>"
}
```
